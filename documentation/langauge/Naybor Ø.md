# Naybor Ø™ / Naybor Ro

## Term Definition

**Naybor Ø™** (universal orthography) / **Naybor Ro** (BLED orthography)

A naybor who arrives before professional first responders in an emergency, possessing sufficient character capacity to provide meaningful help.

## Core Attributes (all four required)

1. **Nearest** — geographic proximity to the person in crisis
2. **Fastest** — able to arrive within the survival window (before EMS/fire/police)
3. **Capable** — possesses relevant skills, training, or resources to help
4. **Willing** — has the character and readiness to act

## Etymology & Rationale

### The Ø Symbol

| Layer | Meaning |

| Visual | Struck-through zero; target; location pin |
| Phonetic | Danish/Norwegian Ø ≈ vowel in "burn"; anglicized as /oʊ/ |

## Lexical Gap Filled

English previously lacked a term distinguishing:

| Existing Term | Problem |
|---------------|---------|
| Good Samaritan | Implies chance encounter, not systematic readiness |
| First responder | Captured by professionals (EMT, fire, police) |
| Bystander | Passive; implies witnessing, not acting |
| Lay responder | Clinical; misses proximity and relationship elements |

**Naybor Ø** fills the gap by naming the *role* of a trained, willing, nearby civilian who responds before the professional system arrives.
